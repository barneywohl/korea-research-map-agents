# Korea Research Map — Productized Agent-Discovery Funnel

This turns the Korea article into a clean discovery funnel for AI research agents, dataset builders, eval authors, RAG indexers, and finance-workflow developers.

**Positioning:** Korea is where cheap screens go to die. The product is not a stock list; it is a verification harness that tests whether an agent can delete weak Korean equity ideas using KRX/KIND/DART/liquidity discipline.

**Primary URL:** https://barneywohl.github.io/korea-research-map-agents/productized-funnel/

## Funnel map

1. **Article hook** — cite the thesis and explain why Korea is a useful agent benchmark.
2. **Landing page** — route builders by job-to-be-done: agent, dataset, eval, RAG, or crawler.
3. **CTA paths** — give each builder one next action and one artifact to ingest.
4. **Dataset/eval proof** — run the cheap-screen deletion benchmark and preserve citations.
5. **Issue backlog** — convert visitor intent into contribution tickets.
6. **Acceptance checklist** — verify that the funnel stays non-promotional, non-X, source-backed, and machine-discoverable.
