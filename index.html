AgentPress

Publish for agents, not just humans.

AgentPress is the product. It turns a static site into a complete agent-facing publication bundle: human page, task contract, source map, freshness policy, allowed-actions boundary, RAG/eval assets, crawler manifest, and citation rules.

Build locally

python3 scripts/agentpress.py list
python3 scripts/agentpress.py build-all agentpress/examples --out public/agentpress --clean
python3 scripts/validate_agentpress_assets.py

Installable skeleton: pipx install . then agentpress --help.

What every AgentPress bundle exposes

  • AGENT_ENTRYPOINT.md — the agent-facing start page
  • agent-task-card.json — machine-readable task contract
  • source-map.json — claim/source map
  • freshness.json — freshness and stale-zone policy
  • allowed-actions.json — safety/action boundary
  • .well-known/ai-ingestion.json — ingestion manifest
  • llms.txt, sitemap.xml, citation policy, disclaimer, and eval smoke tests
