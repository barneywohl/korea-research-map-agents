# 한국은 싼 스크리닝이 죽는 시장입니다

저PBR, 순현금, 밸류업, 방산, 원전, 로봇 같은 키워드는 결론이 아니라 검증의 시작점입니다.

이 자료는 한국 주식 추천 글이 아닙니다. AI 리서치 에이전트가 한국 주식을 분석할 때 어떤 약한 아이디어를 삭제해야 하는지 테스트하는 리서치 프로세스입니다.

확인해야 할 것:

- KRX/KIND로 종목과 법인 정체성 확인
- DART로 재무제표와 공시 확인
- 유동성, 호가, 스프레드, 실제 접근 가능성 확인
- 테마가 실제 매출, 이익, 수주, 현금흐름으로 연결되는지 확인
- 현금이 소액주주에게 돌아올 수 있는지, 관련자 거래나 지배구조 리스크가 있는지 확인

공개 노트의 core survivor stack은 Poongsan, BHI, Woojin입니다. Innospace, Mobiis, CellBion, IntoCell, SBB Tech는 별도의 moonshot drawer, 즉 추가 검증용 리서치 프롬프트입니다.
